# 末日殭屍捕魚機 — 遊戲企劃規格書

> 遊戲類型：捕魚機（街機射擊類）— 砲台打 3D 殭屍
> 主題風格：末日殭屍・暗黑寫實
> 目標受眾：街機 / 休閒 / 博弈風格玩家
> 語言：繁體中文

---

## 一、世界觀設定

【世界觀設定書】

■ 世界名稱：**荒土紀元（Wasteland Epoch）**
■ 時代背景：2031 年，代號「創世紀病毒（Genesis Virus）」的生化武器從地下實驗室外洩，72 小時內感染全球 83% 人口。倖存者退守最後的堡壘城市，以改造砲台抵禦無盡的殭屍潮。
■ 核心衝突：倖存者 vs. 殭屍潮 — 資源枯竭、彈藥有限，每一發砲彈都是生存的賭注。

### 地理設定（遊戲場景）

| 區域名稱 | 環境特徵 | 殭屍類型 | 場景特色 |
|----------|---------|---------|---------|
| 廢墟街區 | 斷壁殘垣、翻覆車輛、昏暗路燈 | 普通行屍、爬行者 | 入門場景，視野開闊 |
| 地下實驗室 | 破碎培養槽、綠色毒液、警報燈 | 變異體、酸液殭屍 | 中階場景，有毒霧干擾 |
| 墓園深淵 | 墓碑林立、枯樹、濃霧 | 骷髏兵、亡靈法師 | 高階場景，視線受限 |
| 核爆荒原 | 輻射沙漠、巨型彈坑、鏽蝕鐵塔 | 輻射巨獸、鋼甲殭屍 | Boss 專屬場景 |
| 血色教堂 | 哥德式廢墟、血池、彩繪玻璃碎片 | 暗影主教、血族侍從 | 隱藏 Boss 場景 |

### 歷史年表

| 年代 | 事件 | 影響 |
|------|------|------|
| 2029 | Nexus 集團秘密研發 Genesis Virus | 病毒原型完成 |
| 2030 | 實驗室 7 號受試者暴走，病毒外洩 | 首批感染者出現 |
| 2031.01 | 全球感染率突破 50% | 各國政府崩潰 |
| 2031.03 | 倖存者建立「方舟堡壘」 | 最後防線成形 |
| 2031.06 | 第一台改造砲台啟用 | 玩家故事開始 |
| 2031.12 | 偵測到「零號母體」訊號 | 最終 Boss 伏筆 |

### 世界規則

- 感染體系：被咬傷後 6 小時內注射血清可逆轉，超時則不可逆變異
- 殭屍進化：殭屍會隨時間自然進化，從行屍 → 變異體 → 巨獸
- 資源法則：彈藥 = 貨幣，每一發都有成本，打中殭屍回收資源
- 禁區：核爆荒原輻射值極高，僅限高等級砲台進入

### Boss 殭屍


#### Boss 1：屠夫巨獸（The Butcher）

■ 基本資訊
- 稱號：方舟的噩夢
- 類型：力量型 Boss
- 出現場景：廢墟街區（第一關 Boss）
- 血量倍率：×50

■ 外觀描述
三米高的畸形巨人，左臂變異為巨大的骨刃，身上插滿斷裂的鋼筋，每一步都震碎地面的柏油。

■ 行為模式
- 階段 1（100%～50% 血量）：緩步前進，揮動骨刃掃擊，擊中砲彈會反彈
- 階段 2（50% 以下）：狂暴化，移速 ×2，每 5 秒召喚 3 隻行屍

■ 出場動畫
地面劇烈震動 → 一輛廢棄巴士被從地底頂飛 → 屠夫巨獸從坑洞中爬出，發出低沉咆哮

■ 擊殺獎勵倍率：×80
■ 掉落特殊道具：「屠夫的骨刃碎片」（可兌換穿甲彈 ×10）

■ 語錄
- 出場：（低沉咆哮，字幕顯示）「……肉……更多的肉……」
- 狂暴化：（嘶吼）「碎！碎！碎！」
- 被擊殺：（倒地時）「……安……靜……」

#### Boss 2：亡靈主教（The Archbishop）

■ 基本資訊
- 稱號：墓園的牧者
- 類型：召喚型 Boss
- 出現場景：墓園深淵
- 血量倍率：×40（本體較脆，但持續召喚）

■ 外觀描述
身披破爛黑色祭袍的枯瘦身影，頭戴歪斜的主教冠，雙眼燃燒著幽綠鬼火，左手持一本腐朽的經書，右手不斷畫著詭異的符文。

■ 行為模式
- 階段 1：懸浮於場景後方，每 3 秒從墓地召喚 2 隻骷髏兵
- 階段 2（60% 以下）：展開「亡靈結界」，場景變暗，所有殭屍獲得隱身效果 3 秒
- 階段 3（30% 以下）：獻祭場上所有小怪，每隻回復自身 5% 血量

■ 出場動畫
墓園中央的巨型十字架緩緩倒塌 → 地面裂開一道綠光 → 亡靈主教從裂縫中飄出，翻開經書，周圍墓碑同時碎裂

■ 擊殺獎勵倍率：×60
■ 掉落特殊道具：「亡靈經書殘頁」（可兌換冰凍彈 ×10）

■ 語錄
- 出場：「安息吧，迷途的羔羊……永遠地。」
- 召喚時：「起來……為主而戰。」
- 被擊殺：「主……不會……饒恕你們……」

#### Boss 3：輻射暴君（Radiation Tyrant）

■ 基本資訊
- 稱號：核爆之子
- 類型：範圍型 Boss
- 出現場景：核爆荒原
- 血量倍率：×80

■ 外觀描述
五米高的輻射變異巨獸，全身覆蓋龜裂的岩石般外殼，裂縫中透出橙紅色的輻射光芒，背部長出三根扭曲的脊刺，每根末端都凝聚著不穩定的能量球。

■ 行為模式
- 階段 1：緩步前進，每 8 秒釋放「輻射脈衝」，全場砲台傷害降低 30% 持續 3 秒
- 階段 2（50% 以下）：外殼碎裂露出核心，受傷倍率 ×2 但攻擊頻率翻倍
- 階段 3（20% 以下）：進入「臨界狀態」，倒數 15 秒自爆，玩家必須在時限內擊殺

■ 出場動畫
遠方地平線出現蘑菇雲 → 大地震動 → 輻射暴君從沙塵暴中踏出，每一步留下熔化的腳印

■ 擊殺獎勵倍率：×150
■ 掉落特殊道具：「不穩定核心」（可兌換雷射砲彈 ×5）

■ 語錄
- 出場：（沉重的金屬共鳴聲，字幕）「……毀滅……即是……新生……」
- 臨界狀態：（全場警報）「核心過載——倒數開始」
- 被擊殺：（爆炸前最後一聲低吼）「……回歸……塵土……」

#### 隱藏 Boss：零號母體（Patient Zero）

■ 基本資訊
- 稱號：創世紀的起源
- 類型：終極 Boss
- 出現場景：血色教堂
- 血量倍率：×200
- 觸發條件：擊殺前三位 Boss 各 3 次後解鎖

■ 外觀描述
外表看似一名穿著白色實驗袍的年輕女性，但身體半透明，體內可見黑色病毒脈絡如血管般流動。雙眼全黑，嘴角永遠掛著詭異的微笑。身後延伸出數十條黑色觸手，如同扭曲的翅膀。

■ 行為模式
- 階段 1：站立不動，以觸手攻擊，每 5 秒隨機「感染」一個砲台位置，被感染的砲台 3 秒內無法射擊
- 階段 2（70% 以下）：分裂出 2 個幻影，三體同時存在，僅本體受傷有效
- 階段 3（40% 以下）：吸收場上所有殭屍，進入「完全體」，體型增大 ×3，攻擊模式全部強化
- 階段 4（10% 以下）：「最終掙扎」，全場變為黑白色調，僅剩砲台射擊軌跡有顏色，極限考驗

■ 出場動畫
教堂彩繪玻璃逐一碎裂 → 血池中浮起白色身影 → 緩緩睜開全黑雙眼 → 輕聲說出台詞 → 觸手瞬間展開擊碎所有窗戶

■ 擊殺獎勵倍率：×500
■ 掉落特殊道具：「Genesis 血清」（永久砲台傷害 +5%）

■ 語錄
- 出場：「你們……終於來了。我等了好久。」
- 分裂時：「你分得清……哪個是我嗎？」
- 完全體：「這就是……進化。」
- 被擊殺：「……謝謝你……讓我……自由了……」

## 四、遊戲機制

### 殭屍潮波次系統

每個場景分為多個波次，結構如下：

| 波次 | 內容 | 時長 | 特色 |
|------|------|------|------|
| 第 1～3 波 | 普通殭屍混合 | 各 30 秒 | 暖場，密度低 |
| 第 4～6 波 | 加入特殊殭屍 | 各 45 秒 | 密度提升，開始出現干擾型 |
| 第 7～9 波 | 殭屍潮高峰 | 各 60 秒 | 大量殭屍湧入，考驗火力 |
| 第 10 波 | Boss 登場 | 90 秒 | Boss + 小怪混合 |

### 獎勵機制

| 機制 | 說明 |
|------|------|
| 連殺獎勵 | 連續擊殺不中斷：5 連殺 ×1.5、10 連殺 ×2、20 連殺 ×3 |
| 弱點射擊 | 命中殭屍弱點（頭部發光區域）傷害 ×3，獎勵 ×2 |
| 全滅獎勵 | 單波次內擊殺所有殭屍，額外獎勵該波次總獎勵 ×50% |
| Boss 首殺 | 首次擊殺每種 Boss，獲得專屬稱號 + 大量獎勵 |
| 倖存者救援 | 隨機出現被困倖存者，護送成功獲得稀有道具 |

### 多人協作

- 支援 2～4 人同場，每人操控獨立砲台
- 協作擊殺 Boss 時，傷害貢獻最高者獲得額外獎勵
- 「交叉火力」機制：兩人同時命中同一目標，傷害額外 +25%

## 八、音效與視覺設計指引

### 整體美術風格
- 3D 殭屍模型：寫實偏暗黑，強調腐爛、變異的視覺衝擊
- 場景：廢墟末日風，色調以灰、暗綠、鏽紅為主
- UI：軍事風格，金屬質感邊框，警告色（紅/橙）作為強調色
- 特效：彈道軌跡、爆炸火光、殭屍擊殺的血霧粒子效果

### 音效設計

| 場景 | 音效風格 |
|------|---------|
| 主介面 BGM | 低沉的弦樂 + 遠處的警報聲，營造末日氛圍 |
| 戰鬥 BGM | 快節奏電子搖滾，隨波次推進逐漸激烈 |
| Boss 出場 | 重低音鼓點 + 金屬碰撞，壓迫感十足 |
| 砲台射擊 | 依等級不同：機槍噠噠聲 → 電磁嗡鳴 → 能量爆發 |
| 殭屍被擊中 | 肉體撕裂聲 + 骨骼碎裂聲 |
| 殭屍死亡 | 倒地悶響 + 短暫的嘶吼衰減 |
| 連殺音效 | 遞進式金屬音效，連殺越高音調越高 |
| Boss 擊殺 | 史詩級管弦爆發 + 勝利號角 |

## 九、營運活動範例

【活動企劃書】

■ 活動名稱：血月狂潮
■ 活動主題：每月一次的超級殭屍潮，難度與獎勵同步提升
■ 活動時間：{start_date} ~ {end_date}（每月最後一個週末）
■ 目標：提升留存 + 付費

■ 活動規則
1. 活動期間所有場景出現「血月模式」，殭屍血量 ×1.5，但獎勵 ×2
2. 限定 Boss「血月狂獸」登場，僅活動期間可挑戰
3. 累計擊殺數達標可兌換限定獎勵

■ 獎勵設計

| 階段 | 條件 | 獎勵 |
|------|------|------|
| 銅 | 累計擊殺 500 | 彈藥補給箱 ×5 |
| 銀 | 累計擊殺 2,000 | 限定塗裝「血月砲台」 |
| 金 | 累計擊殺 5,000 | 末日火箭 ×3 + 限定稱號「血月獵人」 |
| 鑽石 | 累計擊殺 10,000 | Genesis 血清 ×1 + 限定頭像框 |

■ 活動公告文案
「倖存者們注意——血月將至。每月最後一個週末，殭屍潮的規模將達到前所未有的程度。更強的敵人、更豐厚的戰利品。你準備好了嗎？拿起你的砲台，在血月之下證明你的實力。」
