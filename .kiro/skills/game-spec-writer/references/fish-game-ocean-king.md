## 3. 核心玩法機制

### 3.1 Core Loop

```
投幣/押注 → 選擇砲台 → 瞄準射擊 → 捕獲魚群 → 獲得積分 → 繼續射擊或結算
```

### 3.2 操作方式

| 操作 | 說明 | 輸入方式 |
|------|------|---------|
| 投幣/上分 | 插入硬幣或紙鈔啟動遊戲 | 投幣器 / 紙鈔機 |
| 選擇武器 | 從多種網槍/砲台中選擇 | 按鈕切換 |
| 瞄準 | 調整砲台方向對準目標 | 搖桿（Joystick） |
| 射擊 | 發射網彈捕捉魚類 | SHOOT 按鈕 |
| 調整倍率 | 調整砲彈積分/下注大小 | +/- 按鈕 |
| 結算 | 按住 Payout 鍵 3 秒領取獎勵 | PAYOUT 按鈕 |

### 3.3 子彈物理

- 子彈碰到螢幕邊緣會反彈，直到擊中魚為止
- 不同砲台的網格大小和威力不同
- 押注越大，砲彈威力越強，捕捉率越高

### 3.4 魚群行為

- 魚群以不同速度和路徑游過螢幕
- 大型魚/Boss 通常移動較慢但需要更多子彈
- 部分魚種有特殊移動路徑（Z字形、迴旋等）
- 場景會定期切換，帶來不同魚群組合


## 10. 營運與合規規範

### 10.1 合規要求

- 派彩百分比需符合當地博弈法規
- 機台需通過 BMM、GLI 或當地認證
- 營運商需取得合法執照
- 隨機數生成器（RNG）需經第三方驗證

### 10.2 負責任博弈

- 投注限制功能
- 遊玩時間提醒
- 自我排除機制
- 年齡驗證機制

### 10.3 品質標準

- 機台通過電氣安全認證（CE/UL）
- 軟體通過公平性測試
- 硬體通過耐用性測試
