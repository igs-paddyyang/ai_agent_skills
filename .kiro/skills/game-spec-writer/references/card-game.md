# 棋牌規格指南（Card Game Spec Guide）

本文件為 game-spec-writer 的棋牌專用指南。產出棋牌規格時載入此文件。

## 章節側重

棋牌的 9 章節框架中，以下章節需要特別展開：

| 章節 | 棋牌側重 |
|------|---------|
| 3. 符號/角色與分值 | 牌面組合賠率表、莊家/閒家規則 |
| 4. 特殊功能 | 側注（Side Bet）、保險、對子獎勵 |
| 5. 投注與倍率 | 投注區域、限紅範圍、賠率表 |
| 6. RTP | 莊家優勢（House Edge）、各投注區 RTP |

## 棋牌類型

| 類型 | 莊家優勢 | 特色 | 複雜度 |
|------|---------|------|--------|
| 百家樂 | 1.06%-1.24% | 簡單規則，高速遊戲 | 低 |
| 21 點 | 0.5%-2% | 策略性高，可算牌 | 中 |
| 德州撲克 | 2%-5% | 玩家對戰，心理博弈 | 高 |
| 龍虎 | 2.56%-3.73% | 極簡規則，快速開牌 | 極低 |
| 三公 | 3%-5% | 傳統玩法，比大小 | 低 |

## 百家樂規格參考

### 基本賠率表

| 投注區域 | 賠率 | 莊家優勢 | RTP |
|---------|------|---------|-----|
| 莊（Banker） | 1:0.95（扣 5% 佣金） | 1.06% | 98.94% |
| 閒（Player） | 1:1 | 1.24% | 98.76% |
| 和（Tie） | 1:8 | 14.36% | 85.64% |
| 莊對（Banker Pair） | 1:11 | 5.27% | 94.73% |
| 閒對（Player Pair） | 1:11 | 5.27% | 94.73% |

### 補牌規則

莊家和閒家的補牌規則是百家樂的核心機制，需在規格文件中完整列出：

| 閒家前兩張點數 | 動作 |
|--------------|------|
| 0-5 | 補牌 |
| 6-7 | 停牌 |
| 8-9 | 天牌，雙方停牌 |

莊家補牌規則依閒家第三張牌而定（完整對照表需在規格文件中展開）。

### 路紙系統

百家樂的路紙（Roadmap）是重要的 UI 元素：

| 路紙 | 說明 | 用途 |
|------|------|------|
| 大路 | 記錄每局莊/閒/和結果 | 基本趨勢 |
| 大眼仔 | 分析大路的規律性 | 進階分析 |
| 小路 | 分析前兩列的規律 | 進階分析 |
| 曱甴路 | 分析前三列的規律 | 進階分析 |
| 珠盤路 | 彩色珠子記錄 | 快速瀏覽 |

## 21 點規格參考

### 基本賠率表

| 結果 | 賠率 | 說明 |
|------|------|------|
| 一般贏 | 1:1 | 點數大於莊家 |
| Blackjack | 3:2 | 首兩張為 A + 10 點牌 |
| 保險 | 2:1 | 莊家明牌為 A 時可買 |
| 投降 | 退回 50% | 部分規則允許 |

### 策略表

規格文件應包含基本策略表（Hard / Soft / Split），標註為「最佳數學策略參考」。

## 通用棋牌 RTP 參考

| 遊戲 | 主注 RTP | 側注 RTP | 說明 |
