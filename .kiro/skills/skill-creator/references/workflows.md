# 工作流程模式

## 循序工作流程

對於複雜任務，將操作拆解為清晰的循序步驟。建議在 SKILL.md 開頭提供流程概覽：

```markdown
填寫 PDF 表單包含以下步驟：

1. 分析表單（執行 analyze_form.py）
2. 建立欄位對應（編輯 fields.json）
3. 驗證對應（執行 validate_fields.py）
4. 填寫表單（執行 fill_form.py）
5. 驗證輸出（執行 verify_output.py）
```

## 條件式工作流程

對於具有分支邏輯的任務，引導 AI 通過決策點：

```markdown
1. 判斷修改類型：
   **建立新內容？** → 遵循下方「建立工作流程」
   **編輯現有內容？** → 遵循下方「編輯工作流程」

2. 建立工作流程：[步驟]
3. 編輯工作流程：[步驟]
```
