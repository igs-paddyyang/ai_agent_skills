Role: Senior Frontend & Data Visualization Expert
Task: Create a professional, premium-looking single-file Dashboard (index.html).

Technical Stack:
- CSS: Tailwind CSS (via CDN)
- JS: Chart.js (via CDN)
- Typography: Inter font from Google Fonts

Design Specifications:
- Theme: Modern SaaS Dashboard, light gray background (#f8fafc) with white cards.
- Colors: Main (#2563eb), Success (#10b981), Warning (#f59e0b), Danger (#ef4444).
- Layout:
  1. Top: Header with title "Clawd Analytics Dashboard" and current date.
  2. Row 1: 4 KPI Cards (Title, Value, Trend badge, Icon).
  3. Row 2: 
     - Left (66%): Smooth Line Chart for "User & Revenue Trends".
     - Right (33%): Doughnut Chart for "Revenue by Category".
  4. Row 3: Full-width Table for "Recent Transactions" with zebra stripes and status badges.

Data Integration:
- The page MUST fetch data from the local endpoint `/api/data` on page load.
- Implement an `updateDashboard()` function that populates all cards, charts, and tables.
- Add a "Refresh" button that triggers a manual update.
- Charts should use `tension: 0.4` for smooth curves and appropriate gradients.

Interactivity:
- Hover effects on cards and table rows.
- Responsive design (stacked layout on mobile).
- Tooltips enabled for all charts.
