## 1. 遊戲概述

### 1.1 Elevator Pitch

多人即時射擊捕魚機，使用砲台射擊海洋生物賺取積分，配合 Boss 戰與迷你遊戲創造高倍率回報。

### 1.2 基本資訊

| 欄位 | 內容 |
|------|------|
| 遊戲名稱 | 捕魚機海王（Ocean King / Sea King） |
| 系列版本 | Ocean King 2: Ocean Monster Plus、Ocean King 3、Ocean King Jackpot |
| 類型 | 博弈射擊（Arcade Fish Hunting） |
| 平台 | 街機機台（Arcade Cabinet）、線上版（Mobile/Web） |
| 遊玩人數 | 4 / 6 / 8 / 10 人同時遊玩 |
| 開發商 | IGS（International Games System）及各授權製造商 |
| 目標受眾 | 成人博弈娛樂、電子遊藝場 |

### 1.3 核心賣點（USP）

- **多人即時競技**：最多 10 人同時射擊，支援合作與競爭模式
- **豐富特殊功能**：炸彈蟹、雷射蟹、閃電連鎖、漩渦魚等多元迷你遊戲
- **Boss 戰系統**：史詩級 Boss 角色提供超高倍率回報
- **視覺震撼**：3D 海底場景、炫彩光效、大螢幕沉浸體驗

## 2. 核心玩法（Core Gameplay）

### 2.1 核心循環（Core Loop）

```
投幣/押注 → 選擇砲台 → 瞄準射擊 → 捕獲魚群 → 獲得積分 → 繼續射擊或結算
```

### 2.2 主要操作

| 操作 | 說明 | 輸入方式 |
|------|------|---------|
| 投幣/上分 | 插入硬幣或紙鈔啟動遊戲 | 投幣器 / 紙鈔機 |
| 選擇武器 | 從多種網槍/砲台中選擇 | 按鈕切換 |
| 瞄準 | 調整砲台方向對準目標 | 搖桿（Joystick） |
| 射擊 | 發射網彈捕捉魚類 | SHOOT 按鈕 |
| 調整倍率 | 調整砲彈積分/下注大小 | +/- 按鈕 |
| 結算 | 按住 Payout 鍵 3 秒領取獎勵 | PAYOUT 按鈕 |

### 2.3 遊戲流程

1. 投入硬幣或鈔票啟動遊戲
2. 從多種網槍中選擇武器，可隨時切換
3. 使用搖桿將砲台對準目標魚群
4. 按下 SHOOT 按鈕射擊
5. 努力捕捉魚群，尤其是高價值魚種
6. 善用隨機出現的迷你遊戲和特殊功能
7. 按住 Payout 按鈕 3 秒即可領取獎勵

### 2.4 子彈機制

- 子彈碰到螢幕邊緣會**反彈**，直到擊中魚為止
- 不同砲台的網格大小和威力不同
- 押注越大，砲彈威力越強，捕捉率越高
- 未擊中魚的子彈不會浪費（反彈機制）

## 5. 砲台與倍率系統（Cannon & Multiplier System）

### 5.1 砲台等級

| 砲台等級 | 網格大小 | 威力 | 每發成本 |
|---------|---------|------|---------|
| 一級砲 | 小 | 低 | 1 分 |
| 二級砲 | 中 | 中 | 5 分 |
| 三級砲 | 大 | 高 | 10 分 |
| 四級砲 | 特大 | 極高 | 50 分 |
| 超級砲 | 全場 | 最高 | 100 分 |

### 5.2 倍率機制

- 基礎倍率由砲台等級決定
- 捕獲魚種的分值 = 魚種基礎倍率 × 砲台當前押注
- 出現「海王」魚種時，倍率可從 4x 瞬間提升到 300x
- 特殊功能（Boss、迷你遊戲）可額外疊加倍率

## 6. RTP 與機率系統（Return to Player & Probability）

### 6.1 RTP 分析（以 Ocean King Jackpot 為例）

| 項目 | RTP |
|------|-----|
| 基礎遊戲 | 94.5% |
| 固定彩金 | 1.2% |
| 累進彩金 | 0.3% |
| 免費暗黑漩渦觸發 | 1.0% |
| **最高總 RTP** | **97.0%** |

### 6.2 波動性

- Ocean King Jackpot 屬於**低波動性**（Low Volatility），容易贏得組合獎勵但單次獎勵較小
- 傳統街機版屬於**高波動性**（High Volatility），大獎高但風險也高

### 6.3 遊戲週期

| 週期 | 描述 | 策略建議 |
|------|------|---------|
| 吃分期 | 系統收取積分，即便小魚也難以捕捉 | 使用小砲，減少押注 |
| 吐分期 | 系統送出積分，相對容易得分 | 使用大砲，加大押注 |
| 休閒期 | 收支大致平衡 | 正常遊玩 |

### 6.4 營運商可調參數

| 參數 | 說明 |
|------|------|
| 派彩百分比 | 營運商可調整整體派彩比例 |
| 高賠率 / 低賠率 | High Odds / Low Odds 設定，影響魚種分值 |
| 難度等級 | 影響整體捕獲難易度 |

> ⚠️ 以上數值需經營運商實際設定與監管機構審核確認。
