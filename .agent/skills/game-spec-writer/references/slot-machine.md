# 老虎機規格指南（Slot Machine Spec Guide）

本文件為 game-spec-writer 的老虎機專用指南。產出老虎機規格時載入此文件。

## 章節側重

老虎機的 9 章節框架中，以下章節需要特別展開：

| 章節 | 老虎機側重 |
|------|-----------|
| 3. 符號與分值 | 符號設計（Wild / Scatter / Bonus / 一般符號）、賠付表 |
| 4. 特殊功能 | Free Spins、Bonus Game、Gamble Feature、累進彩金 |
| 5. 投注與倍率 | 賠付線數、投注範圍、最大贏獎倍數 |
| 6. RTP | 基礎遊戲 RTP + Free Spins RTP + Bonus RTP + 總 RTP |

## 老虎機類型

| 類型 | 軸數 | 賠付線 | 特色 |
|------|------|--------|------|
| 經典三軸 | 3 | 1-5 | 簡單直覺，高頻小獎 |
| 五軸視頻 | 5 | 9-50 | 主流類型，豐富特效 |
| Megaways | 6 | 最高 117,649 | 動態賠付線，高波動 |
| 累進彩金 | 5 | 20-25 | 連線彩金池，超高獎金 |
| Cluster Pays | 5-7 | 無固定線 | 群組消除，連鎖反應 |

## 符號設計參考

### 標準符號配置（五軸 25 線為例）

| 符號類型 | 數量 | 功能 | 賠付範圍 |
|---------|------|------|---------|
| Wild | 1 | 替代所有符號（除 Scatter） | 5x: 1000x |
| Scatter | 1 | 觸發 Free Spins（3+ 個） | 3x: 5x, 4x: 20x, 5x: 100x |
| Bonus | 1 | 觸發 Bonus Game | 不直接賠付 |
| 高賠符號 | 4 | 主題相關圖案 | 5x: 100x-500x |
| 低賠符號 | 4 | 撲克牌 A/K/Q/J | 5x: 20x-80x |

### 賠付表範例（以投注額為基準）

| Wild | 50x | 200x | 1000x |
| 高賠 A | 30x | 100x | 500x |
| 高賠 B | 25x | 80x | 300x |
| 高賠 C | 20x | 60x | 200x |
| 高賠 D | 15x | 40x | 100x |
| 低賠 A | 5x | 15x | 80x |

## RTP 參考值

| 組成 | RTP 範圍 | 說明 |
|------|---------|------|
| 基礎遊戲 | 85%-90% | 一般轉動的回報 |
| Free Spins | 3%-8% | 免費旋轉的額外回報 |
| Bonus Game | 1%-3% | 獎勵遊戲的回報 |
| 累進彩金 | 1%-2% | 彩金池貢獻 |
| 總 RTP | 92%-97% | 業界標準範圍 |

## 波動性分類

| 波動性 | Hit Rate | 最大贏獎 | 適合玩家 |
|--------|----------|---------|---------|
| 低 | 35%-40% | 500x-1000x | 休閒玩家 |
| 中 | 25%-30% | 2000x-5000x | 一般玩家 |
| 高 | 15%-20% | 10000x-50000x | 高風險偏好 |
