## 8. 自訂魚種設計指南

當使用者需要設計新的魚種時，參考以下設計原則：

### 8.1 倍率設計原則

| 設計要素 | 建議 |
|---------|------|
| 體型與倍率正相關 | 越大的魚倍率越高 |
| 速度與難度正相關 | 越快的魚越難捕捉 |
| 出現頻率與倍率反相關 | 高倍率魚出現頻率低 |
| Boss 需有多階段 | 至少 2-3 個攻擊模式 |
| 特殊效果需平衡 | 高倍率特殊效果應有低觸發率 |

### 8.2 魚種設計模板

```markdown
| 欄位 | 值 |
|------|------|
| 中文名 | [名稱] |
| 英文名 | [Name] |
| 分類層級 | Tier [1-5] |
| 基礎倍率 | [數值]x |
| 倍率範圍 | [最低]x - [最高]x |
| 出現頻率 | 極高/高/中/低/極低/極稀有 |
| 難度星級 | ★☆☆☆☆ ~ ★★★★★ |
| 移動速度 | 慢/中慢/中/快/極快 |
| 移動路徑 | 直線/曲線/Z字形/迴旋/隨機 |
| 特殊效果 | [無/觸發連鎖/觸發漩渦/自帶技能] |
| 視覺描述 | [外觀描述] |
| 設計理念 | [為什麼要有這條魚] |
```

### 8.3 平衡性檢查清單

- [ ] Tier 1 魚種佔總出現數的 60% 以上
- [ ] Tier 2 魚種佔 20-30%
- [ ] Tier 3 魚種佔 5-10%
- [ ] Tier 4（Boss）佔 2-5%
- [ ] Tier 5（超級）佔 < 1%
- [ ] 總 RTP 落在目標範圍內
- [ ] 各層級倍率沒有重疊交叉（避免混淆）
- [ ] 視覺上能清楚區分不同層級
