# 捕魚機規格指南（Fish Game Spec Guide）

本文件為 game-spec-writer 的捕魚機專用指南。產出捕魚機規格時載入此文件。

## 章節側重

捕魚機的 9 章節框架中，以下章節需要特別展開：

| 章節 | 捕魚機側重 |
|------|-----------|
| 3. 符號/角色與分值 | 魚種分值表（Tier 1-5 分層，含倍率、難度、出現頻率） |
| 4. 特殊功能 | 迷你遊戲（閃電鏈、炸彈蟹、旋風等）、Boss 戰 |
| 5. 投注與倍率 | 砲台系統（等級、成本、威力、網格大小） |
| 6. RTP | 基礎遊戲 RTP + 彩金 RTP + 總 RTP 分解 |

## 業界參考

- 海王系列（Ocean King / Sea King）為業界標準
- 詳細規格參考 `fish-game-ocean-king.md`
- 魚種資料庫參考 `fish-game-species-db.md`

## 版本差異

| 版本 | 重點 | 特有內容 |
|------|------|---------|
| Ocean King 2 | 經典模式 | 17 種魚 + 4 Boss + 6 種迷你遊戲 |
| Ocean King 3 | 進化版 | 更多魚種、新 Boss、新武器 |
| Ocean King Jackpot | 線上版 | RTP 97%、累進彩金、低波動性 |
