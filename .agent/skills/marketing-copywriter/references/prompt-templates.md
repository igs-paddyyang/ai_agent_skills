# 行銷文案 Prompt 模板

## Facebook 廣告文案模板

```
請為以下產品撰寫 Facebook 廣告文案：
- 產品：{產品描述}
- 目標受眾：{受眾}
- 產出 3 個版本：理性版、感性版、急迫版
- 每版包含：Hook 標題、本文（80-150 字）、CTA、Hashtag
```

## AIDA 公式模板

```
請使用 AIDA 公式撰寫文案：
- Attention（注意）：用一句話抓住目光
- Interest（興趣）：描述痛點或需求
- Desire（慾望）：展示解決方案與效益
- Action（行動）：明確的 CTA

產品：{產品描述}
```

## 內容日曆模板

```
請為以下品牌產出 7 天社群內容排程：
- 品牌：{品牌描述}
- 平台：{平台}
- 每天包含：發布時間、文案、Hashtag、配圖建議
- 內容類型混搭：知識分享、案例故事、互動問答、產品介紹
```
