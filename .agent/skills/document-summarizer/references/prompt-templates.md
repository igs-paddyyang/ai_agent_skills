# 文件摘要 Prompt 模板

## 基礎摘要模板

```
請閱讀以下文件，並產出結構化摘要：

1. 一句話摘要（不超過 30 字）
2. 核心重點（3-5 項，依重要性排序）
3. 行動建議（2-3 項，含優先序）
4. 關鍵數據（如有）

文件內容：
{貼入文件}
```

## 主管版摘要模板

```
請為以下文件產出「主管版摘要」：
- 限制在一頁 A4 以內
- 重點用粗體標示
- 行動建議標註負責部門
- 省略技術細節，聚焦商業影響

文件內容：
{貼入文件}
```

## 比較摘要模板

```
請比較以下 {N} 份文件的核心差異：
- 產出比較表格
- 標示各文件的獨特觀點
- 總結共識與分歧

文件 1：{貼入}
文件 2：{貼入}
```
