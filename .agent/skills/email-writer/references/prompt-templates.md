# Email 撰寫 Prompt 模板

## 客訴回應模板

```
請撰寫一封回覆客戶投訴的 Email：
- 語氣：致歉 + 專業
- 結構：致歉 → 原因說明 → 補償方案 → 後續追蹤承諾
- 收件人：{客戶名稱}
- 投訴內容：{描述}
```

## 合作邀約模板

```
請撰寫一封商務合作邀約 Email：
- 語氣：正式 + 友善
- 結構：自我介紹 → 合作效益 → 具體提案 → 會議邀請
- 收件人：{對象}
- 合作內容：{描述}
```

## 多版本產出模板

```
請針對以下情境，同時產出「正式版」與「友善版」兩種 Email：
- 情境：{描述}
- 收件人：{對象}
- 目的：{目的}
```
