# 技能知識網路（Skill Tapestry）

> 建立技能間的關聯索引與知識網路，管理依賴、互補、替代與延伸關係，根據情境推薦技能組合。

## 功能說明

建立技能間的關聯索引與知識網路。核心功能包含四種關聯類型定義（依賴、互補、延伸、替代）、自動掃描 .kiro/skills/ 產生關聯圖、技能分類體系（7 大分類）、情境推薦（根據工作情境推薦技能組合），以及 JSON/Markdown 雙格式索引輸出。

## 使用方式

觸發此技能的方式：

```
「幫我建立技能關聯索引」
「列出所有技能的關係」
「我現在要做 X，推薦用哪些技能？」
「更新技能地圖」
「技能總覽」
```

## 檔案結構

```
.kiro/skills/skill-tapestry/
├── SKILL.md                          # 主要技能指令（關聯類型、掃描流程、情境推薦）
├── README.md                         # 本文件
└── references/                       # 詳細指南（按需載入）
    └── tapestry-format.md            # 關聯索引格式規範（JSON/Markdown）與進階用法
```

## 設計參考

本技能參考 [awesome-claude-skills](https://github.com/ComposioHQ/awesome-claude-skills) 的 tapestry 概念，
針對本專案的 Kiro Skills 體系（19 個技能）設計關聯管理機制。
