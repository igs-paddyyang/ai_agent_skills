# 遊戲類型章節側重指南

不同遊戲類型在 GDD 中有不同的重點章節。產出 GDD 時，根據遊戲類型加深對應章節的內容深度。

---

## 類型側重總覽

| 類型 | 重點章節 | 可簡化章節 | 特有內容 |
|------|---------|-----------|---------|
| 動作（Action） | 核心玩法、系統機制 | 敘事 | 操作手感、打擊回饋、連招系統 |
| RPG | 系統機制、敘事、關卡 | — | 技能樹、裝備系統、任務分支、對話系統 |
| 策略（Strategy） | 系統機制、關卡 | 美術（相對） | 資源管理、科技樹、AI 策略、勝利條件 |
| 模擬（Simulation） | 系統機制、關卡 | 敘事 | 模擬精度、時間系統、經營數值 |
| 解謎（Puzzle） | 核心玩法、關卡 | 敘事、技術 | 謎題設計原則、難度遞進、提示系統 |
| 博弈（Gambling/Slot） | 系統機制、營運 | 敘事、關卡 | RTP、賠率表、中獎機率、免費遊戲觸發、彩金系統 |
| 休閒（Casual） | 核心玩法、營運 | 技術、敘事 | 單局時長、社交機制、每日任務、廣告整合 |
| 多人線上（MMO/Multiplayer） | 系統機制、技術、營運 | — | 網路架構、同步機制、社交系統、公會、PvP/PvE |

---

## 各類型詳細指南

### 動作（Action）

重點展開：
- **核心玩法**：操作手感描述（輸入延遲、動畫取消、緩衝輸入）、連招系統、閃避/格擋機制
- **系統機制**：傷害計算公式、無敵幀、擊退距離、攻擊速度

額外表格：
- 招式表（招式名稱、輸入指令、傷害、硬直幀數）
- 敵人行為模式表（敵人類型、攻擊模式、弱點、掉落物）

### RPG

重點展開：
- **系統機制**：角色屬性（力量/敏捷/智力等）、技能樹結構、裝備詞綴系統、經驗值曲線
- **敘事**：主線劇情分章、支線任務設計、對話選項與分支影響
- **關卡**：世界地圖區域劃分、地城設計、Boss 戰設計

額外表格：
- 角色屬性成長表（等級 vs 各屬性值）
- 裝備稀有度表（白/綠/藍/紫/橙，屬性範圍）
- 技能樹結構（分支、前置條件、效果）

### 策略（Strategy）

重點展開：
- **系統機制**：資源類型與產出、科技樹、單位克制關係、勝利條件
- **關卡**：地圖生成規則、地形效果、戰略要點

額外表格：
- 單位屬性表（單位、HP、攻擊、防禦、移動速度、克制關係）
- 科技樹（科技名稱、前置條件、研究時間、效果）
- 資源產出表（建築、產出資源、產出速率）

### 模擬（Simulation）

重點展開：
- **系統機制**：模擬精度與簡化取捨、時間流速控制、經營數值平衡
- **關卡**：場景/地圖的可互動元素、解鎖條件

額外表格：
- 建築/設施表（名稱、成本、產出、解鎖條件）
- 時間事件表（觸發條件、事件內容、影響）

### 解謎（Puzzle）

重點展開：
- **核心玩法**：謎題設計原則（教學→應用→變化→組合）、提示系統設計
- **關卡**：難度遞進曲線、謎題類型分類、每關新機制引入節奏

額外表格：
- 謎題類型表（類型、機制描述、首次出現關卡）
- 難度曲線表（關卡區間、新機制、預估完成時間）

### 博弈（Gambling / Slot）

重點展開：
- **系統機制**：RTP（Return to Player）、賠率表、中獎機率分布、免費遊戲觸發條件、彩金（Jackpot）系統
- **營運**：合規要求、負責任博弈機制、地區法規差異

額外表格：
- 賠率表（符號組合、賠率倍數、出現機率）
- RTP 分析表（基礎遊戲 RTP、免費遊戲 RTP、總 RTP）
- 彩金等級表（等級、觸發條件、獎金範圍）

特別注意：
- 所有機率數值需精確到小數點後 4 位
- 標註「以上數值需經數學模型驗證與監管機構審核」

### 休閒（Casual）

重點展開：
- **核心玩法**：單局時長控制（建議 1-3 分鐘）、上手難度（30 秒內理解規則）
- **營運**：每日任務、成就系統、社交分享、廣告整合點、內購設計

額外表格：
- 每日任務表（任務類型、完成條件、獎勵）
- 內購商品表（商品、價格、效果、是否影響平衡）

### 多人線上（MMO / Multiplayer）

重點展開：
- **技術**：網路架構（Client-Server / P2P）、同步機制、延遲補償、反作弊
- **系統機制**：社交系統（好友、公會、交易）、PvP 匹配、PvE 副本設計
- **營運**：伺服器架構、版本更新策略、賽季系統

額外表格：
- 伺服器架構表（區域、容量、同步方式）
- PvP 段位表（段位、分數範圍、獎勵）
- 副本難度表（難度、建議等級、建議人數、掉落物）
