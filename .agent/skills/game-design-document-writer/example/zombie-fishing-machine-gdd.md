## 1. 遊戲概述（Overview）

### 1.1 Elevator Pitch

末日世界中，玩家操控改造砲台射擊殭屍潮，每發砲彈都是成本，擊殺殭屍回收資源——結合捕魚機的射擊手感與殭屍末日的暗黑美學。

### 1.2 基本資訊

| 欄位 | 內容 |
|------|------|
| 遊戲名稱 | 末日殭屍：最後防線（Dead Zone: Last Stand） |
| 類型 | 捕魚機（街機射擊）/ 博弈風格 |
| 平台 | 手遊（iOS / Android）、網頁（HTML5）、街機 |
| 目標受眾 | 18-45 歲，街機射擊愛好者、休閒博弈玩家 |
| 預估遊玩時長 | 單局 5-10 分鐘，每日 30-60 分鐘 |
| 靈感來源 | 捕魚達人、殭屍末日題材、House of the Dead |

### 1.3 核心賣點（USP）

- 捕魚機玩法 × 殭屍末日主題，市場差異化明確
- 4 個 Boss（含隱藏 Boss「零號母體」），每個有多階段行為模式與專屬演出
- 砲台 5 級升級 + 6 種特殊彈藥，策略深度超越傳統捕魚機
- 多人協作（2~4 人）+ 交叉火力加成機制

## 2. 核心玩法（Core Gameplay）

### 2.1 核心循環（Core Loop）

```
投入彈藥（成本）→ 瞄準射擊殭屍 → 擊殺回收資源（獎勵）→ 升級砲台/購買特殊彈藥 → 挑戰更高難度場景
```

這是典型的捕魚機經濟循環：每一發砲彈都有成本，擊殺目標回收的資源大於或小於成本取決於目標類型與玩家技巧。高價值目標（Boss、稀有殭屍）風險高但回報豐厚。

### 2.2 主要操作

| 操作 | 說明 | 輸入方式 |
|------|------|---------|
| 瞄準 | 控制砲台方向 | 觸控拖曳 / 滑鼠移動 / 搖桿 |
| 射擊 | 發射砲彈，消耗彈藥幣 | 點擊（單發）/ 長按（連射） |
| 切換砲台等級 | 調整彈藥成本與傷害 | 雙指縮放 / 滾輪 / 按鈕 |
| 使用特殊彈藥 | 切換至特殊彈藥類型 | 畫面下方快捷欄點擊 |

### 2.3 進程系統

- 累計擊殺數解鎖更高等級砲台（Lv.1 → Lv.5）
- 通關場景解鎖下一場景（廢墟街區 → 地下實驗室 → 墓園深淵 → 核爆荒原）
- 擊殺前三位 Boss 各 3 次解鎖隱藏場景「血色教堂」
- 成就系統提供長期目標（15 個成就）

### 2.4 難度曲線

| 場景 | 難度定位 | 殭屍血量範圍 | 建議砲台等級 |
|------|---------|-------------|-------------|
| 廢墟街區 | 入門 | ×1 ~ ×4 | Lv.1 ~ Lv.2 |
| 地下實驗室 | 中階 | ×1.5 ~ ×6 | Lv.2 ~ Lv.3 |
| 墓園深淵 | 高階 | ×1.8 ~ ×8 | Lv.3 ~ Lv.4 |
| 核爆荒原 | 極難 | ×4 ~ ×80 | Lv.4 ~ Lv.5 |
| 血色教堂 | 終極 | ×6 ~ ×200 | Lv.5 + 特殊彈藥 |

> ⚠️ 以上數值為建議值，需經實際測試調整。

## 3. 系統機制（Game Systems）

### 3.1 經濟系統

本遊戲的核心經濟為「彈藥 = 貨幣」，每一發砲彈都有成本，擊殺殭屍回收資源。

| 貨幣 | 獲取方式 | 用途 | 說明 |
|------|---------|------|------|
| 彈藥幣 | 擊殺殭屍、波次結算、活動獎勵 | 射擊消耗、砲台升級 | 核心貨幣，流通量最大 |
| 兌換材料 | Boss 掉落 | 兌換特殊彈藥 | 稀有貨幣，綁定 Boss |
| 鑽石 | 付費購買、成就獎勵 | 商城購買、加速 | 付費貨幣 |


#### 經濟循環模型

```
玩家投入彈藥幣 → 射擊殭屍
  ├─ 命中：回收 擊殺獎勵倍率 × 砲台等級 的彈藥幣
  ├─ 未命中：損失該發砲彈成本
  └─ Boss 擊殺：大量彈藥幣 + 兌換材料
```

#### 預估 RTP（Return to Player）

| 項目 | RTP 範圍（建議值） |
|------|-------------------|
| 基礎射擊（普通殭屍） | 85% ~ 92% |
| Boss 戰 | 95% ~ 110%（高風險高回報） |
| 特殊彈藥使用 | 90% ~ 105% |
| 活動期間（血月狂潮） | 95% ~ 115% |
| 總體 RTP | 90% ~ 96% |

> ⚠️ 以上 RTP 為設計目標，需經數學模型驗證。實際 RTP 取決於玩家行為與砲台選擇。

- 消耗道具「急救血清」可立即冷卻

## 4. 關卡與內容（Level & Content）

### 4.1 關卡結構

線性場景解鎖 + 隱藏場景分支。每個場景包含 10 個波次，第 10 波為 Boss 戰。

```
廢墟街區（入門）→ 地下實驗室（中階）→ 墓園深淵（高階）→ 核爆荒原（極難）
                                                                    ↓
                                              血色教堂（隱藏，擊殺前三 Boss 各 3 次解鎖）
```

### 4.2 場景設計

| 場景 | 環境特徵 | 殭屍類型 | 場景特色 |
|------|---------|---------|---------|
| 廢墟街區 | 斷壁殘垣、翻覆車輛、昏暗路燈 | 行屍、爬行者 | 入門場景，視野開闊 |
| 地下實驗室 | 破碎培養槽、綠色毒液、警報燈 | 變異體、酸液殭屍 | 毒霧干擾視線 |
| 墓園深淵 | 墓碑林立、枯樹、濃霧 | 骷髏兵、亡靈法師 | 濃霧限制視線 |
| 核爆荒原 | 輻射沙漠、巨型彈坑、鏽蝕鐵塔 | 輻射巨獸、鋼甲殭屍 | 輻射干擾砲台 |
| 血色教堂 | 哥德式廢墟、血池、彩繪玻璃碎片 | 暗影主教、血族侍從 | 隱藏 Boss 專屬 |

### 4.3 波次系統

每個場景分為 10 個波次：

| 波次 | 內容 | 時長 | 特色 |
|------|------|------|------|
| 第 1～3 波 | 普通殭屍混合 | 各 30 秒 | 暖場，密度低 |
| 第 4～6 波 | 加入特殊殭屍 | 各 45 秒 | 密度提升，干擾型登場 |
| 第 7～9 波 | 殭屍潮高峰 | 各 60 秒 | 大量殭屍湧入 |
| 第 10 波 | Boss 登場 | 90 秒 | Boss + 小怪混合 |

### 4.4 可重玩性

- 每次進入場景，殭屍出現位置與組合隨機生成
- Boss 掉落道具有機率性，鼓勵重複挑戰
- 連殺紀錄排行榜驅動競爭
- 月度活動「血月狂潮」提供限定內容

## 5. 敘事與世界觀（Narrative & World）

### 5.1 故事大綱

2029 年，跨國企業 Nexus 集團秘密研發代號「Genesis Virus」的生化武器。2030 年，實驗室 7 號受試者暴走，病毒外洩。72 小時內感染全球 83% 人口，各國政府在數週內崩潰。

2031 年初，倖存者退守最後的堡壘城市「方舟堡壘」，以改造砲台抵禦無盡的殭屍潮。玩家扮演方舟堡壘的砲台操作員，在資源枯竭的末日中，用每一發砲彈換取生存的機會。

隨著殭屍潮的推進，玩家逐漸深入病毒的源頭——從廢墟街區到地下實驗室，從墓園深淵到核爆荒原，最終在血色教堂面對一切的起源：零號母體（Patient Zero）。擊敗她，不是結束，而是理解這場災難真正意義的開始。

### 5.2 角色設定

本遊戲無傳統意義上的主角角色，玩家身份為「方舟堡壘砲台操作員」。核心角色為殭屍敵人：

| 角色 | 身份 | 在故事中的角色 |
|------|------|--------------|
| 屠夫巨獸 | 早期感染的建築工人 | 第一道考驗，象徵暴力的末日 |
| 亡靈主教 | 感染前的教區神父 | 信仰崩壞的隱喻 |
| 輻射暴君 | 核電廠事故中的變異體 | 人類科技反噬的象徵 |
| 零號母體 | Nexus 實驗室 7 號受試者 | 一切的起源，悲劇的核心 |

### 5.3 世界觀

**世界名稱**：荒土紀元（Wasteland Epoch）

**核心衝突**：倖存者 vs. 殭屍潮——資源枯竭、彈藥有限，每一發砲彈都是生存的賭注。

**世界規則**：
- 感染體系：被咬傷後 6 小時內注射血清可逆轉，超時不可逆變異
- 殭屍進化：殭屍隨時間自然進化，行屍 → 變異體 → 巨獸
- 資源法則：彈藥 = 貨幣，每一發都有成本，擊殺殭屍回收資源
- 禁區：核爆荒原輻射值極高，僅限高等級砲台進入

**歷史年表**：

| 年代 | 事件 | 影響 |
|------|------|------|
| 2029 | Nexus 集團秘密研發 Genesis Virus | 病毒原型完成 |
| 2030 | 實驗室 7 號受試者暴走，病毒外洩 | 首批感染者出現 |
| 2031.01 | 全球感染率突破 50% | 各國政府崩潰 |
| 2031.03 | 倖存者建立「方舟堡壘」 | 最後防線成形 |
| 2031.06 | 第一台改造砲台啟用 | 玩家故事開始 |
| 2031.12 | 偵測到「零號母體」訊號 | 最終 Boss 伏筆 |

### 5.4 主題

生存的代價——在末日中，每一個選擇都有成本。砲彈是有限的，殭屍是無盡的。遊戲透過「彈藥 = 貨幣」的核心機制，讓玩家在每一次射擊中感受資源管理的壓力與擊殺的爽快之間的張力。

## 6. 美術風格（Art & Visual）

### 6.1 視覺風格

3D 寫實偏暗黑風格，強調腐爛、變異的視覺衝擊。整體色調以灰、暗綠、鏽紅為主，營造壓抑的末日氛圍。光源以昏暗的環境光為基底，搭配砲彈軌跡、爆炸火光等高亮特效形成視覺對比。

### 6.2 角色設計方向

**殭屍設計原則**：
- 普通殭屍：保留人類輪廓但明顯腐爛變異，每種類型有清晰的視覺辨識度
- Boss：體型明顯大於普通殭屍，設計語言強調其類型特徵（力量型→巨大肌肉、召喚型→飄浮祭袍、範圍型→發光裂縫）
- 零號母體：刻意與其他殭屍形成反差——外表接近人類，詭異感來自半透明身體與黑色脈絡

**弱點視覺**：所有殭屍頭部有微弱的發光區域，提示玩家弱點位置。
