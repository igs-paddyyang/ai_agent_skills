# 簡報生成 Prompt 模板

## 提案簡報模板

```
請為以下主題產出提案簡報大綱：
- 主題：{主題}
- 目標受眾：{受眾}
- 結構：問題 → 分析 → 方案 → 效益 → 時程 → 預算
- 頁數：10-15 頁
- 每頁包含：標題、核心訊息、內容重點（3項）、視覺建議、講者備註
```

## 管理層月報模板

```
請產出管理層月報簡報大綱：
- 結構：KPI 總覽 → 本月亮點 → 待解決問題 → 下月計畫
- 頁數：6-10 頁
- 重點用數據說話，每頁建議一個圖表
```

## Marp 格式模板

```
請以 Marp Markdown 格式產出投影片：
- 主題：{主題}
- 使用 --- 分隔每頁
- 包含 YAML 前置設定（theme、paginate、backgroundColor）
- 每頁使用 ## 作為標題
```
